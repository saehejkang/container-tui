# container-tui

A terminal-based user interface (TUI) for managing `apple/container`.

> [!IMPORTANT]
> Features, structure, and behavior are still under active development and may change frequently.

## Requirements

- `apple/container` [CLI](https://github.com/apple/container) installed and setup

## Build & Usage

Build the executable and run the TUI:

```bash
go get
go build
./container-tui
```

## Technologies Used

- **Go** – programming language
- **[Bubble Tea](https://github.com/charmbracelet/bubbletea)** – TUI framework
- **[Lip Gloss](https://github.com/charmbracelet/lipgloss)** – styling for terminal output
